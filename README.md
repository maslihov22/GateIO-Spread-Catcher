# Spread Catcher

Скринер для поиска арбитражных возможностей между спотовым и фьючерсным рынками на Gate.io.

## Что делает

- Сканирует все USDT пары с perpetual futures на Gate.io (~300+ пар)
- Вычисляет спред между спотом и фьючерсами
- Рассчитывает профитность с учетом комиссий (0.2% спот + 0.05% фьючерс)
- Мониторит funding rate для funding arbitrage
- Обновляется каждые 10 секунд

## Стратегии

**Price Spread Arbitrage** - когда фьючерс дороже спота:
- Купить на споте
- Зашортить на фьючерсах
- Закрыть когда спред сузится

**Funding Rate Arbitrage** - при положительном funding rate:
- Та же позиция (спот лонг + фьючерс шорт)
- Получаете funding каждые 8 часов (3 раза в день)
- APY может достигать 50%+ на волатильных монетах

```json
{

}
```

```bash
npm start
```

## Параметры

| Параметр | Значение | Описание |
|----------|----------|----------|
| CAPITAL | $200 | Базовый капитал для расчета ($100 спот + $100 фьючерс) |
| MIN_PROFITABLE_SPREAD | 0.6% | Минимальный чистый спред для сигнала |
| MIN_APY | 5% | Минимальный APY для funding арбитража |

## Комиссии (заложены в расчет)

- Spot: 0.2% (taker)
- Futures: 0.05% (taker)
- Round-trip: ~0.5%
